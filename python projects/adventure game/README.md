# Adventure Game

This folder contains the adventure game scripts and assets.

- `adventure_game.py` is the text adventure version.
- `adventure_game_gui.py` is the Tkinter GUI version.
- `csv_to_pixels.py` converts editable CSV artwork into drawing code.
- `game photos` holds the image, sprite, and scene asset files.
- The `grid_*.csv` and `forest_stars_*.csv` files store scene layouts and pixel data.
