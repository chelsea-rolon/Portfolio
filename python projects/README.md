# Python Projects

This folder collects small apps, experiments, and supporting assets.

- `adventure game` contains the text and GUI adventure game scripts, CSV scene assets, and game artwork.
- `karaoke sign up` contains the FastAPI karaoke sign-up app, admin page, playlists, and performer media.
- `expense tracker stuff` contains the bank statement processor web app and its supporting notebook work.
- Other files and folders here are shared support files and generated artifacts.
