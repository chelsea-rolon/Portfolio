Project Details

Choose Your Quest Game (CLI + GUI)

A narrative adventure implemented in Python with both terminal and Tkinter GUI versions, supported by custom game assets and scenes.

Back to Projects

Overview

This project explores story-driven gameplay and branching experiences with a reusable Python codebase and visual assets. Users can play as two different characters, and multiple endings are available based on player choices.

Tech Stack

  • Language: Python
  • Interface: CLI + Tkinter GUI
  • Assets: Sprite, overlays, and scene files

Scene Photos

  • Features sixteen playable scenes and twenty one sprite configurations.
  • Two different playable characters
  • Multiple endings based on player choices
