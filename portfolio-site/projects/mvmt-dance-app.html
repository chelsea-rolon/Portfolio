MVMT Dance Class App

A responsive dance marketplace prototype inspired by real booking pain points, focused on intuitive UX and mobile-first interaction.

Back to Projects

Overview

Many of my friends who are in the dance comminutiy struggle to find a consisitnet way to discover classes, apply for gigs, and engage with instructors. MVMT helps dancers by providing a consistent interface for all these activities.

Tech Stack

  • Frontend: HTML5, CSS3, JavaScript
  • Architecture: Multi-page prototype flow
  • Design Focus: Responsive UX and discoverability
  • GitHub Link

UX Highlights

  • Designed a mobile-first interface for discovering and booking dance classes.
  • Implemented intuitive navigation flows for both dancers and instructors.
  • Focused on clean, modern aesthetics to enhance user engagement.
