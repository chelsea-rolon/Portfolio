# MVMT App

A polished dance marketplace prototype for discovering nearby classes, booking classes and gigs, uploading dancer videos, and letting instructors publish classes.

## Features

- Nearby class discovery with filters for skill level, cost, length, radius, and online classes
- Book classes and apply for gigs from inside the app
- Dancer profile with bookings, saved classes, and video uploads
- Instructor class upload form
- Responsive single-page layout built with vanilla HTML, CSS, and JavaScript

## Run

Open `index.html` in a browser, or serve the folder with any static server.
