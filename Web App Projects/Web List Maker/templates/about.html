About The Web App

A Flask + SQLAlchemy to-do list app where you can create tasks with a title, due date, and details — then mark them complete, edit, or delete them.

About the Creator

This was created by Chelsea Rolon as part of a student web development project.

I took the basic Flask + SQLAlchemy app that I had learned in class and made it more visually appealing and user-friendly by adding styling and enhancing the user interface. In addition, I added a due date feature to help users stay organized.

Thank you for using the app!
